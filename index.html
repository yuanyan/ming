modulejs by yuanyan

modulejs is a modular and Object-Oriented javascript framework that is designed to write a high maintainability and reusability of the javascript code.

Features: HTML document query/traversing/manipulation Lightweight selector engine Event handling Animation Cookie/localStorage storage Language extensions Ajax/JSONP request Template engine Rich utility class
